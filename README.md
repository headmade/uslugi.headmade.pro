# Environment

For work effectively requires the version of `Node.js` from the project.

```sh
nvm use
```

or

```sh
nvm install 
```

## Node modules 

Need to use `npm`

```sh
npm i
```

## Run project

Need to use `npm`

```sh
npm run develop
```
